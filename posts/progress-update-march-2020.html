Progress Update (March 2020)

Published: Thu 05 March 2020
By Jackkel Dragon

In misc.

tags: updates

It’s been a while since I posted a full update here, so I’m going to include a bunch of things that may have already been mentioned elsewhere to catch up a bit.

Updates to Released Games

Last year saw not only the release of Eldritch Academy and the first full version of Blooming Nightshade, but also the release of updates to both games:

For Eldritch Academy, the v2 update was released to the Steam version of the game. This update removed a lot of similar scenes to make replaying easier and trimmed down the first two acts of the game to remove scenes that felt like filler. There have been no bug reports from the Steam version, so it may be released on itch.io as well.

For Blooming Nightshade, the v10 update changed the format of the game to simplify route selection while making each route more unique (including at least one new scene for each route). The former random events are now integrated into specific routes, based on which characters were involved. For those who preferred the original version, v9 is still available on itch.io and as a beta build on Steam.

Upcoming Projects

Eldritch University is coming along well, with almost half of the main game now fully implemented. The price is still in flux, but the game will almost certainly be cheaper than Eldritch Academy. No release date has been set, but I hope to release this year if possible.

Flowering Nightshade recently got its Demo v2 release, which revamped the game to have more linear routes and includes the first full version of Izumi and Nanami’s routes. Kiku’s route is now in development, as is an unrevealed route.

In addition, an ebook project that I’ve been working on for the past year is in its revision stage. Barring any surprises, the ebook will likely be released sometime this year. This story is roughly the same length as Shireishi or h+, and will likely have a similar price point.

And that’s about it for now. Hopefully I’ll have an announcement for Eldritch University or the ebook in the near future, but we’ll see. I don’t want to rush anything out, though I have been tempted a few times. Until then, I hope people continue to enjoy the stories that have been released so far!
